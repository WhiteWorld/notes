对象

Java 的对象是通过引用来访问的。
数据存储位置

- 寄存器，位于处理器内部，Java 程序中感知不到寄存器的存在
- 堆栈，通过堆栈指针访问，对象引用位于堆栈中
- 堆，存放所有 Java 对象
- 常量存储，常量值存放在程序代码内部，不会被改变。保存在已编译的 .class 文件中，存放基本类型和对象型的常量值。还包括文本形式出现的符合引用。
- 非 RAM 存储
